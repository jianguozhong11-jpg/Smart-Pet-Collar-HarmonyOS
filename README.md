Harmony-PetCare：基于鸿蒙分布式的智能宠物项圈系统2024年中国大学生计算机设计大赛 
参赛项目本项目是一款基于 HarmonyOS 开发的智能物联网（IoT）原型系统。通过模拟项圈端与手机端的分布式协同，实现了宠物健康实时监测、智能行为干预及环境联动，旨在利用分布式技术解决城市养宠过程中的焦虑监测与邻里矛盾痛点。
🌟 核心亮点分布式协同（The Soul）：基于 DistributedObject 实现项圈与手机端的毫秒级数据同步，打破设备边界。智能干扰算法：内置焦虑等级评估模型，综合吠叫强度与活动量，自动触发声音/气味干预。人文关怀场景：独创“防止邻里冲突模式”，动态调整告警阈值，平衡宠物安抚与邻里安宁。
🛠️ 技术栈开发语言：ArkTS (HarmonyOS 声明式开发)框架：OpenHarmony / HarmonyOS SDK (API 9+)核心特性：分布式数据对象 (DistributedObject)、通知服务 (Notification)、多媒体服务 (AudioRenderer)、UIAbility 交互。
📦 功能模块全天候健康监测：模拟心率（$BPM > 140$ 自动预警）、体温、呼吸及压力指数。多维行为干预：手机端远程控制项圈释放安抚信息素或播放主人录音。环境智能联动：感知空气质量，如厕后自动触发除味喷雾。场景化管理：离家守望模式与邻里和谐模式切换。
